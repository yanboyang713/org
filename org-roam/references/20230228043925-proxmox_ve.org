* Introduction
:PROPERTIES:
:ID:       86b06445-3f9c-4195-ae29-3f43bb513e4c
:END:
Proxmox VE is a complete, open-source server management platform for enterprise virtualization. Based on [[id:0c65c1a6-4751-4290-876f-6c5ad7694068][Debian]]. It tightly integrates the [[id:a7a76158-d895-4c2a-bb16-1ec4fe46edaa][Kernel Based Virtual Machive (KVM)]] hypervisor and [[id:2c382716-6c15-4385-945f-e5f64e0c0b6e][Linux Containers (LXC)]], software-defined storage and networking functionality (such as,[[id:db3292f7-9665-41fb-bc4d-76ed2be72c99][EVPN]]), on a single platform. With the integrated web-based user interface you can manage VMs and containers, high availability for clusters, or the integrated disaster recovery tools with ease.
