#+title: CachyOS
#+date: 2026-06-28

CachyOS is a Linux distribution based on [[id:dc13b67c-8d8b-40fd-b8cf-9ea8547e485d][Arch Linux]]. It focuses on speed and security optimisations - the default Linux kernel is heavily optimised using the BORE (Burst-Oriented Response Enhancer) scheduler, while the desktop packages are compiled with LTO, x86-64-v3 and x86-64-v4, Zen 4 optimization, security flags and performance improvements. The available desktop environments, window managers and [[id:11743715-9a10-4732-9081-68d0a614cf20][Wayland]] compositors include bspwm, Budgie, Cinnamon, COSMIC, GNOME, [[id:46ff078d-0aa4-42a8-a300-07c444184f27][Hyprland]], i3, KDE Plasma, LXDE, LXQt, MATE, Niri, Openbox, Qtile, Sway, UKUI, Wayfire and Xfce. CachyOS also ships with both graphical and command-line system installers.
