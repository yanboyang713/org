#+title: Hugging Face Accelerate
#+date: 2026-05-18

Accelerate is a popular, easy-to-use library built to work with PyTorch models. It provides a very straightforward API that wraps with PyTorch and transformers code.

Accelerate can automatically allocate GPU memory usage across nodes, and is used for both training and inference. It is built on top of Megatron and can interface with [[id:986e00f4-0df0-4968-a1c1-aa625bfedf6d][DeepSpeed]].
