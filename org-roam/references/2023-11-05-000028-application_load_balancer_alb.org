#+title: Application Load Balancer (ALB)

ALB is best suited for load balancing of HTTP and HTTPS traffic and provides advanced request routing targeted at the delivery of modern application architectures, including microservices and containers.

Operating at the individual request level (Layer 7), Application Load Balancer routes traffic to targets within [[id:7aed7e4c-350e-4322-b5f7-52b860fe75cb][AWS virtual private cloud (VPC)]] based on the content of the request.
