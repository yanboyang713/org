#+title: User Plane Function (UPF)
#+date: 2025-09-16

The User Plane Function (UPF) is the data-plane anchor of the 5G core. It forwards user packets between the radio side and external data networks while applying the forwarding and traffic-handling decisions made by the control plane.

The UPF is commonly associated with:
+ packet forwarding and routing
+ traffic steering and breakout
+ quality-of-service treatment and usage reporting
+ acting as the user-plane anchor for subscriber sessions
