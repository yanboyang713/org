** Change the boot order
You may need to change the boot order to boot from a USB drive or optical media. Menu names can vary depending on whether the system is configured for BIOS or UEFI boot.

1. From the =System Setup= main menu, go to =System BIOS= → =Boot Settings=.
2. Select =Boot Option Settings= → =Boot Sequence=.
3. Use the arrow keys to select a boot device; use =+= and =-= keys to move it up or down.
4. Exit and confirm saving changes.
