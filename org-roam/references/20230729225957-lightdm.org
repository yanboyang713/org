#+title: LightDM
#+filetags:  

LightDM is a cross-desktop [[id:9f8e5ab1-cbcb-4290-a8ca-7941a0a9b821][display manager]]. Its key features are:
+ Cross-desktop - supports different desktop technologies.
+ Supports different display technologies ([[id:fe1f3869-8620-4fad-8b01-f2fa6aa75331][Xorg/X11]], Mir, [[id:11743715-9a10-4732-9081-68d0a614cf20][Wayland]] ...).
+ Lightweight - low memory usage and high performance.
+ Supports guest sessions.
+ Supports remote login (incoming - XDMCP, VNC, outgoing - XDMCP, PAM).
+ Comprehensive test suite.
+ Low code complexity.
